# Hello World

安装好 docker 后，可以执行以下命令来测试。

docker run hello-world
1

# docker 镜像使用

# 搜索镜像

docker search ubuntu
1

# 获取镜像

docker pull ubuntu
1

这会获取 tag 为 lastest 的 ubuntu 镜像，如果要指定 tag，可以在镜像名后附加 :tag

docker pull ubuntu:20.04
1

# 列出镜像列表

docker images
1

# docker 容器使用

# 交互式运行

在使用docker run命令时，通过附加两个参数-i -t可以在启动 docker 容器时进入交互式界面。

两个参数的意义：

  • -t: 在新容器内置顶一个伪终端或终端。
  • -i: 允许你对容器内的标准输入（STDIN）进行交互。

# docker 镜像加速

对于使用 systemd 的系统，在 /etc/docker/daemon.json 中写入如下内容（如果文件不存在则新建该文件）：

{
  "registry-mirrors":["https://hub-mirror.c.163.com/"]
}

sudo systemctl restart docker
